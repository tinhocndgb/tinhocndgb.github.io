Chương trình máy tính là gì?

Giải thích: Chương trình máy tính là tập hợp các lệnh được viết bằng ngôn ngữ lập trình để máy tính thực hiện.

Trình biên dịch (compiler) khác trình thông dịch (interpreter) ở điểm nào?

Giải thích: Compiler dịch toàn bộ chương trình thành mã máy, interpreter dịch và thực thi từng dòng.

Trong Python, lệnh nào dùng để chuyển đổi chuỗi "123" thành số nguyên?

Giải thích: Hàm int() chuyển đổi chuỗi hoặc số thực thành số nguyên.

Kiểu dữ liệu boolean trong Python nhận những giá trị nào?

Giải thích: Kiểu bool trong Python chỉ có hai giá trị: True và False (viết hoa chữ cái đầu).

Toán tử "//" trong Python thực hiện phép toán gì?

Giải thích: // là toán tử chia lấy phần nguyên (floor division).

Cấu trúc điều khiển nào sau đây dùng để xử lý nhiều trường hợp?

Giải thích: Cấu trúc if-elif-else dùng để kiểm tra nhiều điều kiện khác nhau.

Vòng lặp nào thích hợp khi không biết trước số lần lặp?

Giải thích: Vòng lặp while dùng khi số lần lặp phụ thuộc vào điều kiện, không biết trước.

Phương thức nào thêm phần tử vào cuối list trong Python?

Giải thích: append() thêm phần tử vào cuối list, insert() thêm vào vị trí chỉ định.

Tham số của hàm là gì?

Giải thích: Tham số (parameter) là biến được định nghĩa trong khai báo hàm, nhận giá trị khi gọi hàm.

Lỗi thời gian chạy (runtime error) xảy ra khi nào?

Giải thích: Lỗi runtime xảy ra khi chương trình đang chạy, như chia cho 0, truy cập index ngoài phạm vi.

Trong Python, câu lệnh nào để mở file "data.txt" để đọc?

Giải thích: Mode "r" (read) dùng để đọc file, "w" để ghi, "a" để thêm vào cuối.

Độ phức tạp thuật toán đo lường điều gì?

Giải thích: Độ phức tạp thuật toán đo lường thời gian chạy và bộ nhớ cần thiết khi kích thước đầu vào tăng.

Python là ngôn ngữ lập trình hướng đối tượng.
Giải thích: Đúng, Python hỗ trợ cả lập trình hướng đối tượng và lập trình cấu trúc.

Giải thích: Sai, range(5) tạo dãy từ 0 đến 4 (0, 1, 2, 3, 4).
Hàm có thể trả về nhiều giá trị trong Python bằng cách dùng tuple.
Giải thích: Đúng, ví dụ: return a, b sẽ trả về tuple (a, b).
Set trong Python cho phép chứa các phần tử trùng lặp.
Giải thích: Sai, set chỉ chứa các phần tử duy nhất, không trùng lặp.

Đánh dấu Đúng (Đ) hoặc Sai (S) cho các câu sau:

Break trong Python dùng để thoát khỏi hàm.
Giải thích: Sai, break dùng để thoát khỏi vòng lặp, return mới dùng để thoát hàm.
List comprehension là cách viết ngắn gọn để tạo list trong Python.
Giải thích: Đúng, ví dụ: [x**2 for x in range(5)] tạo list [0, 1, 4, 9, 16].
Đệ quy có thể gây tràn bộ nhớ stack nếu không có điều kiện dừng.
Giải thích: Đúng, mỗi lần gọi đệ quy thêm một frame vào stack, có thể gây stack overflow.
Tất cả các thuật toán đều có thể chuyển đổi thành chương trình máy tính.
Giải thích: Sai, chỉ các thuật toán thực thi được (computable) mới có thể lập trình.

Giải thích: ** có độ ưu tiên cao hơn *, nên 3**2=9, 2*9=18.

len("Python") trả về giá trị bao nhiêu?

Giải thích: clear() xóa tất cả phần tử, list trở thành [] (rỗng), len([])=0.

Kết quả của "Hello"[1] trong Python là gì?

Giải thích: Chuỗi index bắt đầu từ 0: "Hello"[0]='H', [1]='e'.

Viết hàm Python tính tổng các số chẵn trong một list số nguyên được truyền vào.

Đáp án Tham khảo:
```python
def tong_so_chan(danh_sach):
  tong = 0
  for so in danh_sach:
    if so % 2 == 0: # Kiểm tra số chẵn
      tong += so
  return tong

# Ví dụ sử dụng:
nums = [1, 2, 3, 4, 5, 6]
print(tong_so_chan(nums)) # Kết quả: 12 (2+4+6)
```

Cách 2 dùng list comprehension:
```python
def tong_so_chan(danh_sach):
  return sum([x for x in danh_sach if x % 2 == 0])
```

Trình bày sự khác biệt giữa tham số và đối số trong hàm Python. Cho ví dụ minh họa.

Đáp án Tham khảo:
- Tham số (Parameter):
  + Là biến được định nghĩa trong khai báo hàm
  + Là "placeholder" cho giá trị sẽ được truyền vào
  + Ví dụ: a, b trong def cong(a, b):

- Đối số (Argument):
  + Là giá trị thực tế được truyền vào hàm khi gọi hàm
  + Là "giá trị thực" thay thế cho tham số
  + Ví dụ: 5, 3 trong cong(5, 3)

Ví dụ minh họa:
```python
# Khai báo hàm với tham số a, b
def cong(a, b): # a, b là tham số
  return a + b

# Gọi hàm với đối số 5, 3
ket_qua = cong(5, 3) # 5, 3 là đối số
print(ket_qua) # Kết quả: 8
```

Viết chương trình Python đọc file "diem.txt" chứa các số điểm (mỗi dòng một số), tính điểm trung bình và lưu kết quả vào file "ketqua.txt".

Đáp án Tham khảo:
```python
# Đọc file diem.txt
with open("diem.txt", "r") as f:
  diem = [float(line.strip()) for line in f if line.strip()]

# Tính điểm trung bình
if len(diem) > 0:
  trung_binh = sum(diem) / len(diem)
else:
  trung_binh = 0

# Ghi kết quả vào file ketqua.txt
with open("ketqua.txt", "w") as f:
  f.write(f"Số điểm: {len(diem)}\n")
  f.write(f"Điểm trung bình: {trung_binh:.2f}\n")
  if trung_binh >= 8:
    f.write("Xếp loại: GIỎI\n")
  elif trung_binh >= 6.5:
    f.write("Xếp loại: KHÁ\n")
  elif trung_binh >= 5:
    f.write("Xếp loại: TRUNG BÌNH\n")
  else:
    f.write("Xếp loại: YẾU\n")

print("Đã xử lý xong!")
```

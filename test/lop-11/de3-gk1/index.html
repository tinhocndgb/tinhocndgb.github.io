Viết hàm Python nhận vào một số nguyên dương n và trả về dictionary chứa bình phương của các số từ 1 đến n.

Đáp án Tham khảo:
```python
def binh_phuong_dict(n):
  """Trả về dictionary {1:1², 2:2², ..., n:n²}"""
  result = {}
  for i in range(1, n + 1):
    result[i] = i ** 2
  return result

# Cách 2 dùng dictionary comprehension:
def binh_phuong_dict(n):
  return {i: i**2 for i in range(1, n + 1)}

# Ví dụ sử dụng:
print(binh_phuong_dict(5))
# Kết quả: {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

Giải thích sự khác nhau giữa shallow copy và deep copy trong Python. Cho ví dụ minh họa.

Đáp án Tham khảo:
- Shallow Copy (bản sao nông):
  + Tạo đối tượng mới nhưng tham chiếu đến cùng các đối tượng con bên trong
  + Thay đổi đối tượng con ảnh hưởng đến cả bản gốc và bản sao
  + Ví dụ: list.copy() hoặc list[:]

- Deep Copy (bản sao sâu):
  + Tạo đối tượng mới và đệ quy sao chép tất cả đối tượng con bên trong
  + Bản sao hoàn toàn độc lập với bản gốc
  + Ví dụ: copy.deepcopy() từ module copy

Ví dụ minh họa:
```python
import copy

# Shallow copy
list1 = [[1, 2], [3, 4]]
list2 = list1.copy()
list2[0][0] = 99
print(list1) # [[99, 2], [3, 4]] - bị thay đổi!

# Deep copy
list3 = [[1, 2], [3, 4]]
list4 = copy.deepcopy(list3)
list4[0][0] = 99
print(list3) # [[1, 2], [3, 4]] - không bị thay đổi
```

Viết chương trình Python tìm ước số chung lớn nhất (USCLN) của hai số nguyên dương sử dụng thuật toán đệ quy.

Đáp án Tham khảo:
```python
def uscln_de_quy(a, b):
  """Tìm USCLN bằng thuật toán Euclid đệ quy"""
  if b == 0:
    return a
  else:
    return uscln_de_quy(b, a % b)

# Nhập dữ liệu
try:
  x = int(input("Nhập số thứ nhất: "))
  y = int(input("Nhập số thứ hai: "))
  if x <= 0 or y <= 0:
    print("Vui lòng nhập số nguyên dương!")
  else:
    ket_qua = uscln_de_quy(x, y)
    print(f"USCLN của {x} và {y} là: {ket_qua}")
    print(f"BSCNN của {x} và {y} là: {x*y//ket_qua}")
except ValueError:
  print("Vui lòng nhập số hợp lệ!")
```

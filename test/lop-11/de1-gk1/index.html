Đề kiểm tra GHK I - Tin học 11 - Đề 1

Ngôn ngữ lập trình bậc cao là gì?

Giải thích: Ngôn ngữ lập trình bậc cao gần với ngôn ngữ tự nhiên, cần chương trình dịch để chuyển sang ngôn ngữ máy.

Chương trình dịch có nhiệm vụ gì?

Giải thích: Chương trình dịch (compiler/interpreter) chuyển mã nguồn sang mã máy để thực thi.

Trong Python, để khai báo một biến số nguyên, ta làm thế nào?

Giải thích: Python là ngôn ngữ động, không cần khai báo kiểu dữ liệu trước.

Kiểu dữ liệu nào trong Python dùng để lưu trữ chuỗi ký tự?

Giải thích: str (string) là kiểu dữ liệu chuỗi trong Python.

Biểu thức nào sau đây trong Python cho kết quả là False?

Giải thích: 10 <= 5 → False, 3 != 3 → False, False or False → False.

Câu lệnh rẽ nhánh nào đúng trong Python?

Giải thích: Python dùng == để so sánh, và kết thúc điều kiện bằng dấu hai chấm.

Vòng lặp for trong Python thường dùng để làm gì?

Giải thích: Vòng lặp for thường dùng để lặp qua các phần tử của một dãy (list, range, string...).

Cấu trúc dữ liệu nào trong Python cho phép lưu trữ nhiều giá trị không thay đổi?

Giải thích: Tuple là cấu trúc dữ liệu bất biến (immutable), không thể thay đổi sau khi tạo.

Hàm trong Python được định nghĩa bằng từ khóa nào?

Giải thích: Trong Python, hàm được định nghĩa bằng từ khóa "def".

Phạm vi của biến cục bộ (local variable) trong Python là gì?

Giải thích: Biến cục bộ chỉ tồn tại và có thể truy cập trong hàm nó được khai báo.

Lỗi cú pháp (syntax error) xảy ra khi nào?

Giải thích: Lỗi cú pháp xảy ra khi vi phạm quy tắc viết code của ngôn ngữ lập trình.

Thuật toán đệ quy là gì?

Giải thích: Thuật toán đệ quy giải quyết bài toán bằng cách gọi lại chính hàm đó với đầu vào nhỏ hơn.

Đánh dấu Đúng (Đ) hoặc Sai (S) cho các câu sau:

Trình thông dịch (interpreter) dịch và thực thi chương trình từng dòng một.
Giải thích: Đúng, trình thông dịch đọc, dịch và chạy từng dòng lệnh.
Trong Python, kiểu dữ liệu của biến phải được khai báo rõ ràng trước khi sử dụng.
Giải thích: Sai, Python là ngôn ngữ động, kiểu dữ liệu được xác định tự động khi gán giá trị.
Vòng lặp while thực hiện các lệnh lặp lại khi điều kiện còn đúng.
Giải thích: Đúng, vòng lặp while kiểm tra điều kiện trước mỗi lần lặp.
List trong Python là cấu trúc dữ liệu không thể thay đổi (immutable).
Giải thích: Sai, list là mutable - có thể thay đổi phần tử.

Đánh dấu Đúng (Đ) hoặc Sai (S) cho các câu sau:

Hàm print() trong Python dùng để nhập dữ liệu từ bàn phím.
Giải thích: Sai, print() dùng để xuất dữ liệu ra màn hình. Nhập dữ liệu dùng input().
Dictionary trong Python lưu trữ dữ liệu dưới dạng cặp key-value.
Giải thích: Đúng, dictionary dùng key để truy cập giá trị tương ứng.
Đệ quy phải có điều kiện dừng, nếu không sẽ gây lặp vô hạn.
Giải thích: Đúng, đệ quy không có điều kiện dừng sẽ gọi hàm vô hạn lần.
Tất cả các ngôn ngữ lập trình đều có cú pháp giống hệt nhau.
Giải thích: Sai, mỗi ngôn ngữ lập trình có cú pháp riêng biệt.

Kết quả của biểu thức 5 ** 2 trong Python là bao nhiêu?

Giải thích: ** là toán tử lũy thừa, 5 ** 2 = 5² = 25.

len([1, 2, 3, 4, 5]) trong Python trả về giá trị bao nhiêu?

Giải thích: Hàm len() trả về số phần tử trong danh sách: 5 phần tử → 5.

Giá trị của x sau khi thực hiện: x = 10; x += 2; x -= 4

Giải thích: x = 10, x += 2 → x=12, x -= 4 → x= 8.

Kết quả của 10 % 3 trong Python là bao nhiêu?

Giải thích: % là toán tử chia lấy dư: 10 chia 3 được 3 dư 1.

Viết chương trình Python kiểm tra một số nguyên nhập vào có phải là số nguyên tố hay không.

Đáp án Tham khảo:
```python
n = int(input("Nhập số nguyên dương: "))
if n <= 1:
  print(n, "không phải số nguyên tố")
else:
  nguyen_to = True
  for i in range(2, int(n**0.5) + 1):
    if n % i == 0:
      nguyen_to = False
      break
  if nguyen_to:
    print(n, "là số nguyên tố")
  else:
    print(n, "không phải số nguyên tố")
```

Phân biệt giữa biến toàn cục (global) và biến cục bộ (local) trong Python. Cho ví dụ minh họa.

Đáp án Tham khảo:
- Biến toàn cục (global):
  + Khai báo bên ngoài hàm
  + Có thể truy cập từ mọi nơi trong chương trình
  + Ví dụ:
  ```python
  x = 10 # Biến toàn cục
  def ham():
    print(x) # In ra 10
  ```

- Biến cục bộ (local):
  + Khai báo bên trong hàm
  + Chỉ tồn tại và truy cập được trong hàm đó
  + Ví dụ:
  ```python
  def ham():
    y = 5 # Biến cục bộ
    print(y) # In ra 5
  # print(y) # Lỗi! y không tồn tại ở đây
  ```

Trình bày thuật toán và viết chương trình Python tính tổng S = 1 + 2 + 3 + ... + N (với N nhập từ bàn phím).

Đáp án Tham khảo:
Thuật toán:
1. Nhập N
2. Khởi tạo tổng S = 0
3. Cho i chạy từ 1 đến N
4. Cộng i vào S: S = S + i
5. Hiển thị kết quả S

Chương trình Python:
```python
# Cách 1: Dùng vòng lặp for
N = int(input("Nhập N: "))
S = 0
for i in range(1, N + 1):
  S += i
print("Tổng là:", S)

# Cách 2: Dùng công thức toán học
N = int(input("Nhập N: "))
S = N * (N + 1) // 2
print("Tổng là:", S)
```

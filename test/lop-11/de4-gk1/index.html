Viết hàm Python kiểm tra xem một chuỗi có phải là palindrome (đối xứng) không, bỏ qua khoảng trắng và phân biệt chữ hoa/thường.

Đáp án Tham khảo:
```python
def is_palindrome(s):
  """Kiểm tra chuỗi có phải palindrome không"""
  # Loại bỏ khoảng trắng và chuyển về chữ thường
  s = ''.join(s.lower().split())
  # So sánh chuỗi với chuỗi đảo ngược
  return s == s[::-1]

# Cách 2: So sánh từng ký tự
def is_palindrome2(s):
  s = ''.join(s.lower().split())
  left, right = 0, len(s) - 1
  while left < right:
    if s[left] != s[right]:
      return False
    left += 1
    right -= 1
  return True

# Ví dụ sử dụng:
print(is_palindrome("A man a plan a canal Panama")) # True
print(is_palindrome("hello")) # False
```

Trình bày sự khác nhau giữa phương thức sort() và hàm sorted() trong Python. Cho ví dụ minh họa.

Đáp án Tham khảo:
- Phương thức sort():
  + Là phương thức của list object
  + Sắp xếp list tại chỗ (in-place)
  + Không trả về list mới (trả về None)
  + Chỉ dùng được với list
  + Cú pháp: list.sort(reverse=False, key=None)

- Hàm sorted():
  + Là built-in function
  + Tạo và trả về list mới đã sắp xếp
  + Không thay đổi iterable gốc
  + Dùng được với mọi iterable (list, tuple, string, dict, set)
  + Cú pháp: sorted(iterable, reverse=False, key=None)

Ví dụ minh họa:
```python
# Sử dụng sort()
numbers = [3, 1, 4, 1, 5, 9]
numbers.sort()
print(numbers) # [1, 1, 3, 4, 5, 9] - list gốc bị thay đổi

# Sử dụng sorted()
numbers = [3, 1, 4, 1, 5, 9]
sorted_numbers = sorted(numbers)
print(numbers) # [3, 1, 4, 1, 5, 9] - list gốc không đổi
print(sorted_numbers) # [1, 1, 3, 4, 5, 9] - list mới

# sorted() với string và tuple
print(sorted("python")) # ['h', 'n', 'o', 'p', 't', 'y']
print(sorted((3, 1, 2))) # [1, 2, 3]
```

Viết chương trình Python đọc file CSV "sinhvien.csv" (có cột: MSSV, Ten, Diem), tính điểm trung bình của lớp, và tìm sinh viên có điểm cao nhất.

Đáp án Tham khảo:
```python
def process_student_csv(filename):
  """Xử lý file CSV sinh viên"""
  try:
    with open(filename, 'r', encoding='utf-8') as f:
      lines = f.readlines()
    
    # Bỏ qua header nếu có
    if lines and (lines[0].lower().startswith('mssv') or ';' in lines[0] or ',' in lines[0]):
      lines = lines[1:]
    
    students = []
    total_score = 0
    max_score = -1
    top_student = ""
    
    for line in lines:
      line = line.strip()
      if not line:
        continue
      
      # Phân tích dòng (giả sử định dạng: MSSV,Ten,Diem)
      parts = line.split(',')
      if len(parts) >= 3:
        mssv = parts[0].strip()
        ten = parts[1].strip()
        try:
          diem = float(parts[2].strip())
        except ValueError:
          continue # Bỏ qua dòng không hợp lệ
        
        students.append({"mssv": mssv, "ten": ten, "diem": diem})
        total_score += diem
        
        # Tìm sinh viên điểm cao nhất
        if diem > max_score:
          max_score = diem
          top_student = ten
  
    if students:
      diem_tb = total_score / len(students)
      print(f"Số sinh viên: {len(students)}")
      print(f"Điểm trung bình lớp: {diem_tb:.2f}")
      print(f"Sinh viên điểm cao nhất: {top_student} ({max_score} điểm)")
    else:
      print("Không có dữ liệu sinh viên")
  
  except FileNotFoundError:
    print(f"Không tìm thấy file {filename}")
  except Exception as e:
    print(f"Lỗi khi xử lý file: {e}")

# Gọi hàm
process_student_csv("sinhvien.csv")
```
